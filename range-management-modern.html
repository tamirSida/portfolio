Shooting Range Management System

A tactical, real-time scheduling and management platform for live-fire shooting ranges, developed for Central Command in the IDF.

Client Central Command, IDF
Type Internal Defense-Tech Web App
Stack React, Firebase, Firestore
Status Operational Deployment

Project Overview

The Range Management System provides full control room capabilities for tactical shooting ranges, enabling real-time monitoring of range states, dynamic unit scheduling, usage analytics, and comprehensive data visualization for operational insights.

This system replaces manual processes with a digital platform that improves safety, optimizes resource allocation, and provides command-level visibility across all training facilities.

Security Notice: Due to the sensitive nature of this project, full source code and certain implementation details are restricted.

Key Features

Real-time Dashboard

Live monitoring of range statuses (Hot / Cold / Pending) with instant updates across all connected devices and control points.

Dynamic Data Aggregation

Automatic data collection by unit, time, and usage frequency for command insights and resource optimization planning.

Range Request System

Secure request and approval workflow with notifications, status tracking, and integrated planning tools for units.

Role-based Access

Customized views and permissions for Admin, Commanders, and Unit Managers with security-first design principles.

Visual Analytics

Interactive dashboard with charts and activity breakdown for data-driven decisions and resource allocation.

Challenges & Solutions

Real-time Synchronization

Challenge: Multiple control points needed access to live range status data with instant updates across secure networks with varying connectivity levels.

Solution: Implemented Firebase's real-time database with optimized listeners and fallback mechanisms for connectivity issues, including offline-first design principles.

Complex Access Control

Challenge: Different user roles required highly specific permissions based on unit, rank, and facilities, with strict security requirements.

Solution: Built a custom RBAC system with hierarchical permission inheritance and context-aware access rules, integrated with military authentication systems.

Legacy System Integration

Challenge: Needed to connect with existing military scheduling systems without disrupting operations or compromising data integrity.

Solution: Developed adapters and middleware services to bridge modern web app with legacy infrastructure, with comprehensive validation and data transformation layers.
