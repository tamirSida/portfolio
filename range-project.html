Shooting Range Management System

Client: Central Command, IDF
Type: Internal Defense-Tech Web App
Stack: React, Firebase, Firestore, TailwindCSS
Status: Operational Deployment

Project Overview

A tactical, real-time scheduling and management platform for live-fire shooting ranges, developed for Central Command in the IDF. The system provides full control room capabilities including range state monitoring, dynamic unit scheduling, usage analytics, and data visualization for operational insights.

Due to the nature of the project, the full source code is available only upon request under NDA.

Key Features

Real-time Dashboard

Live monitoring of range statuses (Hot / Cold / Pending) with instant updates.

Dynamic Data Aggregation

Automatic data collection by unit, time, and usage frequency for command insights.

Range Request System

Secure request and approval workflow with notifications and status tracking.

Role-based Access

Customized views and permissions for Admin, Commanders, and Unit Managers.

Visual Analytics

Interactive dashboard with charts and activity breakdown for data-driven decisions.

Technical Implementation

Frontend

  • React for component-based UI architecture
  • TailwindCSS for custom responsive design
  • Chart.js for interactive data visualizations
  • React Router for multi-view navigation

Backend

  • Firebase Authentication for secure multi-tier access
  • Firestore for real-time data and state management
  • Cloud Functions for request processing and notifications
  • Custom API integrations with military systems

Security Features

  • Role-based access control with granular permissions
  • Secure data transmission with encryption
  • Audit logging for all critical operations
  • Session management with automatic timeouts

Challenges & Solutions

Real-time Synchronization

Multiple control points needed access to live range status data with instant updates across secure networks.

Solution: Implemented Firebase's real-time database with optimized listeners and fallback mechanisms for connectivity issues.

Complex Access Control

Different user roles required highly specific permissions based on unit, rank, and facilities.

Solution: Built a custom RBAC system with hierarchical permission inheritance and context-aware access rules.

Legacy System Integration

Needed to connect with existing military scheduling systems without disrupting operations.

Solution: Developed adapters and middleware services to bridge modern web app with legacy infrastructure.
