Project Overview

Visual range booking and management

Range-Cal was developed to solve the challenge of efficiently scheduling and coordinating the use of military training ranges across multiple bases. The system offers a visual, map-based interface that allows unit commanders to easily identify available ranges, book them according to their training requirements, and receive appropriate approvals through an automated workflow.

Note: Some technical details have been abstracted due to security considerations.

Challenges & Solutions

Overcoming technical hurdles

Complex Mapping Requirements

Challenge: Creating an interactive map that accurately represents physical training ranges with all relevant metadata.
Solution: Implemented custom Leaflet.js extensions with GeoJSON data structures and interactive popups.

Multi-Level Approval Process

Challenge: Designing a flexible workflow system to handle various approval paths based on range type and unit classification.
Solution: Developed a state machine-based workflow engine with configurable approval chains and notification triggers.

Concurrent Booking Conflicts

Challenge: Preventing double bookings while allowing multiple users to view and schedule ranges simultaneously.
Solution: Implemented optimistic UI with server-side validation and real-time locking mechanisms during the booking process.

Mobile Field Usage

Challenge: Ensuring the system works effectively on mobile devices in field conditions with limited connectivity.
Solution: Created a progressive web app with offline capabilities, optimized map tiles, and low-bandwidth mode.
